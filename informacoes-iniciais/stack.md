# Stack do MVP de Ocupação de Leitos (esusreport)

- Infraestrutura: VPS gerenciada via Easypanel com deploy automatizado via GitHub
- Backend: Python com FastAPI
- Tratamento de dados: Pandas
- API REST para consumo do frontend
- Frontend: Next.js com React, TypeScript e Tailwind CSS
- Banco de dados: MySQL compartilhado
- Tabela principal ainda não criada: ocupacao_leitos_esusreport
